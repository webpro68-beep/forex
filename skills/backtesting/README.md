# Backtesting Skills

This package includes components for running simple backtests, calculating performance metrics, and generating equity curves.

Modules:
- `run_backtest.py` — execute a backtest over price and signal history.
- `metrics.py` — calculate return and Sharpe ratio.
- `equity_curve.py` — build a series from positions and prices.
- `schemas.py` — backtest result data model.
