# Risk Management Skills

This package contains utilities for calculating position sizing, risk/reward, stop loss, and take profit levels.

Modules:
- `position_size.py` — determine trade size based on account risk.
- `stop_loss.py` — compute stop loss placement using volatility.
- `take_profit.py` — calculate profit target distances.
- `risk_reward.py` — derive ratio metrics for trade planning.
- `schemas.py` — result models for risk recommendations.
